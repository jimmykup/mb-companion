Tags that support HTML height attribute

This document was built to test which HTML elements support the HTML height="" attribute. The purpose of this is to help build an efficient code validation function that informs the user when they could possibly run into height related bugs in clients like Yahoo! and Outlook 120dpi.

Conclusion

We are only interested in 1) HTML elements that can be given an HTML height attribute so that we can solve the Yahoo! min-height bug in modern desktop web browsers, and 2) elements that Outlook allows us to modify the height of.

The following tags support the HTML height="" attribute in Google Chrome 83: table, thead, tbody, tfoot, tr, th, td, marquee, img, image, input[type='image'], object, embed, video, iframe, and canvas.

The following tags support the HTML height="" attribute in Microsoft Outlook 2007+ on Windows: tr, th, td, img.
